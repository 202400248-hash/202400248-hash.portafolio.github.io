Pisos Del Caribe
Descripción

Pisos del Caribe es una plataforma web desarrollada como catálogo digital y cotizador interactivo para la venta e instalación de pisos vinílicos tipo SPC, duela y laminados.

El sistema permite visualizar modelos de productos, consultar características técnicas y explorar distintas colecciones mediante un catálogo dinámico orientado a la experiencia del cliente.

Integra una calculadora de metros cuadrados que estima automáticamente la cantidad de material necesario, número de cajas y costos aproximados de instalación, optimizando el proceso de cotización de manera rápida y precisa.

El proyecto simula un entorno comercial real, combinando funcionalidades de e-commerce, gestión de productos y herramientas de cálculo enfocadas al sector de recubrimientos para interiores.

Tecnologías utilizadas
  • Frontend: HTML, CSS, JavaScript
  • Framework UI: Bootstrap
  • Backend: PHP
  • Base de datos: MySQL
  • Consultas SQL dinámicas
  • Formularios de cotización

App de Gestión de Tareas
Descripción

ScannerEdu es una interfaz web diseñada para el registro y control de estudiantes mediante escaneo de códigos QR, permitiendo validar accesos de forma rápida, automatizada y segura dentro de un entorno institucional.

El sistema simula el flujo de ingreso a instalaciones educativas, mostrando en tiempo real el estado del escaneo, la verificación del usuario y la confirmación de acceso dentro de la plataforma.

El proyecto se enfocó en la optimización del proceso de registro, buscando reducir tiempos de validación manual y mejorar la organización del control estudiantil mediante soluciones digitales.

Tecnologías utilizadas
  • Diseño de interfaz: Figma
  • UX/UI enfocado en experiencia de escaneo
  • Prototipado web responsive
  • Flujos de validación de usuarios
  • Diseño visual orientado a sistemas institucionales

Dispensario Medico
Descripción

Salud Total es un sistema web desarrollado para la administración del dispensario médico institucional, enfocado en la gestión digital de medicamentos, control de proveedores y administración de accesos de usuarios.

La plataforma permite registrar, consultar, actualizar y eliminar medicamentos mediante operaciones CRUD, facilitando el control del inventario y la disponibilidad de insumos médicos en tiempo real.

Además, integra un sistema de autenticación con sesiones que restringe el acceso únicamente al personal autorizado, garantizando la seguridad de la información y el manejo adecuado de los registros del dispensario.

Tecnologías utilizadas
  • Frontend: HTML, CSS, JavaScript
  • Framework UI: Bootstrap
  • Backend: PHP
  • Base de datos: MySQL
  • Métodos de envío: GET y POST
  • Gestión de sesiones: PHP Sessions
